# 🧬 Computational Drug Discovery Using PCA and K-Means Clustering

## 📌 Project Overview

This project demonstrates the application of Artificial Intelligence (AI) and Machine Learning (ML) techniques in computational drug discovery. Using Python, Principal Component Analysis (PCA), and K-Means Clustering, molecular data was analyzed to identify patterns and group compounds with similar characteristics.

The project was completed as part of the **Artificial Intelligence in Biotechnology and Healthcare** course offered by **BioGrademy**.

---

## 🎯 Objectives

- Analyze molecular compound data using machine learning.
- Reduce dataset dimensionality using Principal Component Analysis (PCA).
- Cluster similar drug compounds using the K-Means algorithm.
- Visualize molecular relationships through two-dimensional plots.
- Demonstrate the role of AI in accelerating computational drug discovery.

## ⚙️ Workflow

1. Load and explore the molecular dataset.
2. Perform data preprocessing.
3. Select numerical features.
4. Standardize the data using `StandardScaler`.
5. Apply Principal Component Analysis (PCA).
6. Perform K-Means clustering.
7. Visualize clusters using scatter plots.
8. Interpret molecular relationships and clustering results.

## 📈 Results

- Successfully reduced high-dimensional molecular data into two principal components.
- Identified distinct clusters of drug compounds using K-Means clustering.
- Generated clear visualizations to illustrate similarities among molecular compounds.
- Demonstrated how AI and machine learning techniques can support computational drug discovery and molecular analysis.
